# Posts Monitor
This is a service to get the posts from [Dcard](https://www.dcard.tw/) and [Reddit](http://www.reddit.com/)

# Techniques
  + Angular.js
  + Node.js

# Routes
  + ```'/'```: HOME page
  + ```'/#/dcard'```: Shows the latest posts within specific category
  + ```'/#/reddit'```: Shows the latest posts
